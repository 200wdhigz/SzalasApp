# Contributing
## 🤝 Jak Kontrybuować
### 1. Fork & Clone
```bash
# Fork na GitHubie, potem:
git clone https://github.com/twoj-user/SzalasApp.git
```
### 2. Utwórz Branch
```bash
git checkout -b feature/moja-funkcja
```
### 3. Commit Changes
```bash
git commit -m "feat: Dodano nową funkcję X
- Opis co zostało dodane
- Dlaczego
- Jak testować"
```
### 4. Push & Pull Request
```bash
git push origin feature/moja-funkcja
# Następnie utwórz PR na GitHubie
```
## Commit Messages
Format: 	ype: krótki opis
**Typy:**
- eat: - Nowa funkcja
- ix: - Bugfix
- docs: - Dokumentacja
- style: - Formatowanie
- efactor: - Refaktoring
- 	est: - Testy
## Code Review
Każdy PR wymaga:
- [ ] Opis zmian
- [ ] Testy (jeśli dotyczy)
- [ ] Aktualizacja dokumentacji
- [ ] Approval od maintainera
