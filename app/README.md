# SzalasApp - System Zarządzania Sprzętem

System do zarządzania sprzętem szczepu z funkcją zgłaszania usterek i logowania przez OAuth.

## Dokumentacja

Pełna dokumentacja projektu znajduje się w głównym pliku [README.md](../README.md) w katalogu głównym repozytorium.

## Szybki Start

Ten pakiet Python zawiera aplikację Flask do zarządzania sprzętem.

### Instalacja zależności

```bash
poetry install
```

### Uruchomienie aplikacji

```bash
poetry run python app.py
```
